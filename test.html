Server Information

Please check your server configuration to ensure PHP is properly installed and configured.

Common Issues:

  • PHP is not installed on the server
  • PHP module is not enabled in the web server configuration
  • File associations for .php files are not set correctly
  • PHP handler is misconfigured

Next Steps:

  1. Contact your hosting provider and ask them to verify PHP is properly configured
  2. Check if your hosting plan includes PHP support
  3. Ask if there's a specific directory where PHP files must be placed
  4. Verify if you need to use a specific file extension (some hosts use .php5 or .php7)
